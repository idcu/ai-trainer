# 11. SPSS安装使用及示例

## 11.1 什么是SPSS

SPSS（Statistical Product and Service Solutions）是IBM公司推出的一款专业的统计分析软件，广泛应用于社会科学、商业研究、医疗卫生、市场调研等领域。它提供了直观的图形用户界面和强大的统计分析功能，即使是没有统计学背景的用户也能快速上手。

### 主要特性
- 友好的图形用户界面，操作简单直观
- 强大的数据管理功能，支持多种数据格式
- 丰富的统计分析方法：描述统计、推断统计、回归分析等
- 强大的数据可视化功能，生成高质量图表
- 支持脚本编程，自动化重复任务
- 广泛的应用场景和完善的学习资源

## 11.2 系统要求

### Windows系统
- 操作系统：Windows 10/11 或 Windows Server 2016/2019/2022
- 处理器：Intel或AMD处理器，至少2GHz
- 内存：至少4GB，推荐8GB或更多
- 磁盘空间：至少2GB可用空间
- 显示器：分辨率至少1024×768

### macOS系统
- 操作系统：macOS 10.15 (Catalina) 或更高版本
- 处理器：Intel或Apple Silicon处理器
- 内存：至少4GB，推荐8GB或更多
- 磁盘空间：至少2GB可用空间
- 显示器：分辨率至少1024×768

## 11.3 安装方式

### 11.3.1 IBM SPSS Statistics正式版安装

#### 步骤1：获取安装包
1. 访问IBM官方网站：https://www.ibm.com/products/spss-statistics
2. 选择适合的版本（Windows/macOS）
3. 下载安装程序
4. 注意：IBM SPSS是商业软件，需要购买许可证

#### 步骤2：安装过程（Windows）
1. 双击下载的安装程序
2. 选择安装语言，点击"确定"
3. 点击"下一步"继续
4. 阅读许可协议，选择"我接受许可协议中的条款"，点击"下一步"
5. 选择安装路径（默认即可），点击"下一步"
6. 选择要安装的组件，点击"下一步"
7. 点击"安装"开始安装
8. 等待安装完成，点击"完成"

#### 步骤3：激活软件
1. 首次启动SPSS时，会弹出激活向导
2. 选择"立即授权产品"
3. 输入许可证代码
4. 按照提示完成激活

### 11.3.2 使用免费试用版

1. 访问IBM SPSS试用页面
2. 注册IBM账号
3. 申请免费试用（通常为30天）
4. 下载并安装试用版

### 11.3.3 使用开源替代方案

如果预算有限，可以考虑以下开源统计软件：
- **PSPP**：GNU项目，与SPSS语法兼容
- **R**：专业的统计编程语言，功能强大
- **Python**：结合Pandas、SciPy、StatsModels等库进行统计分析

## 11.4 SPSS界面介绍

### 11.4.1 数据视图
- 类似于Excel表格，用于输入和查看数据
- 每一行代表一个案例（观察值）
- 每一列代表一个变量

### 11.4.2 变量视图
- 用于定义和修改变量属性
- 可设置变量名、类型、标签、值、缺失值等

### 11.4.3 菜单栏
- **文件**：打开、保存、导入/导出数据
- **编辑**：剪切、复制、粘贴、查找等
- **视图**：切换数据视图/变量视图
- **数据**：数据操作功能
- **转换**：变量转换和计算
- **分析**：各种统计分析方法
- **图形**：数据可视化
- **实用程序**：实用工具
- **窗口**：窗口管理
- **帮助**：帮助文档

### 11.4.4 工具栏
- 提供常用功能的快捷按钮
- 可自定义工具栏

## 11.5 数据管理

### 11.5.1 创建新数据集

#### 步骤1：定义变量
1. 点击"变量视图"标签
2. 输入变量名（例如：性别、年龄、成绩）
3. 选择变量类型：
   - 数值：用于数值型数据
   - 字符串：用于文本数据
   - 日期：用于日期数据
4. 设置变量标签（变量的详细描述）
5. 设置值标签（用于分类变量）

#### 示例：定义学生信息变量

| 名称 | 类型 | 标签 | 值 |
|------|------|------|-----|
| id | 数值 | 学号 | - |
| name | 字符串 | 姓名 | - |
| gender | 数值 | 性别 | 1=男, 2=女 |
| age | 数值 | 年龄 | - |
| score | 数值 | 成绩 | - |

#### 步骤2：输入数据
1. 切换到"数据视图"
2. 在表格中输入数据

### 11.5.2 导入外部数据

#### 导入Excel数据
1. 点击"文件" → "打开" → "数据"
2. 文件类型选择"Excel (*.xls, *.xlsx)"
3. 选择要导入的Excel文件
4. 在导入向导中选择工作表和数据范围
5. 点击"确定"完成导入

#### 导入CSV数据
1. 点击"文件" → "打开" → "数据"
2. 文件类型选择"逗号分隔 (*.csv)"
3. 选择CSV文件
4. 在文本导入向导中设置分隔符
5. 点击"完成"

### 11.5.3 数据筛选

#### 使用选择个案
1. 点击"数据" → "选择个案"
2. 选择选择方式：
   - 全部个案
   - 如果条件满足
   - 随机个案样本
   - 基于时间或个案范围
   - 使用筛选变量
3. 设置条件后点击"确定"

#### 示例：筛选成绩大于80分的学生
```
选择"如果条件满足"
点击"如果"按钮
输入条件：score > 80
点击"继续"
点击"确定"
```

## 11.6 基本统计分析

### 11.6.1 描述统计

#### 频率分析
1. 点击"分析" → "描述统计" → "频率"
2. 将需要分析的变量选入"变量"框
3. 点击"统计量"选择需要的统计指标
4. 点击"图表"选择图表类型
5. 点击"确定"

#### 描述性统计
1. 点击"分析" → "描述统计" → "描述"
2. 将变量选入"变量"框
3. 点击"选项"选择统计指标
4. 点击"确定"

#### 探索分析
1. 点击"分析" → "描述统计" → "探索"
2. 将因变量选入"因变量列表"
3. 将分组变量选入"因子列表"
4. 点击"统计量"和"图"进行设置
5. 点击"确定"

### 11.6.2 交叉表分析

1. 点击"分析" → "描述统计" → "交叉表"
2. 将行变量选入"行"框
3. 将列变量选入"列"框
4. 点击"精确"设置精确检验
5. 点击"统计量"选择卡方检验等
6. 点击"单元格"设置单元格显示
7. 点击"确定"

## 11.8 高级统计分析

### 11.8.1 t检验

#### 独立样本t检验
1. 点击"分析" → "比较均值" → "独立样本t检验"
2. 将检验变量选入"检验变量"
3. 将分组变量选入"分组变量"
4. 点击"定义组"设置分组
5. 点击"确定"

#### 配对样本t检验
1. 点击"分析" → "比较均值" → "配对样本t检验"
2. 将成对变量选入"成对变量"
3. 点击"确定"

### 11.8.2 方差分析

#### 单因素方差分析
1. 点击"分析" → "比较均值" → "单因素ANOVA"
2. 将因变量选入"因变量列表"
3. 将因子变量选入"因子"
4. 点击"事后比较"选择多重比较方法
5. 点击"选项"选择描述统计等
6. 点击"确定"

### 11.8.3 相关分析

#### 皮尔逊相关分析
1. 点击"分析" → "相关" → "双变量"
2. 将变量选入"变量"框
3. 选择相关系数类型
4. 选择显著性检验
5. 点击"确定"

### 11.8.4 回归分析

#### 线性回归
1. 点击"分析" → "回归" → "线性"
2. 将因变量选入"因变量"
3. 将自变量选入"自变量"
4. 选择方法（输入、逐步等）
5. 点击"统计量"选择输出内容
6. 点击"确定"

## 11.9 完整示例：学生成绩分析

### 场景说明
我们有一组学生的考试成绩数据，包括性别、年龄、数学成绩、语文成绩。我们需要进行以下分析：
1. 描述性统计分析
2. 性别对成绩的影响
3. 数学和语文成绩的相关性
4. 建立回归模型

### 步骤1：准备数据

创建数据集，包含以下变量：
- id：学号
- gender：性别（1=男，2=女）
- age：年龄
- math：数学成绩
- chinese：语文成绩

示例数据（前10条）：

| id | gender | age | math | chinese |
|----|--------|-----|------|---------|
| 1  | 1      | 18  | 85   | 78      |
| 2  | 1      | 19  | 72   | 85      |
| 3  | 2      | 18  | 90   | 92      |
| 4  | 2      | 19  | 68   | 75      |
| 5  | 1      | 18  | 76   | 80      |
| 6  | 2      | 19  | 88   | 86      |
| 7  | 1      | 18  | 95   | 88      |
| 8  | 2      | 19  | 65   | 72      |
| 9  | 1      | 18  | 78   | 79      |
| 10 | 2      | 19  | 82   | 90      |

### 步骤2：描述性统计分析

1. 点击"分析" → "描述统计" → "描述"
2. 将math和chinese选入"变量"
3. 点击"选项"，勾选：均值、标准差、最小值、最大值
4. 点击"继续"，点击"确定"

**预期输出**：

| 变量 | N | 最小值 | 最大值 | 均值 | 标准差 |
|------|---|--------|--------|------|--------|
| math | 10 | 65 | 95 | 80.9 | 9.5 |
| chinese | 10 | 72 | 92 | 82.5 | 6.3 |

### 步骤3：性别对成绩的影响

#### 独立样本t检验

1. 点击"分析" → "比较均值" → "独立样本t检验"
2. 将math和chinese选入"检验变量"
3. 将gender选入"分组变量"
4. 点击"定义组"，组1输入1，组2输入2
5. 点击"继续"，点击"确定"

**解读结果**：
- 查看"独立样本检验"表格
- 如果方差齐性检验（Levene检验）的显著性 > 0.05，读取第一行
- 如果显著性 < 0.05，读取第二行
- t检验的显著性（Sig. 2-tailed）< 0.05表示差异显著

### 步骤4：数学和语文成绩的相关性

1. 点击"分析" → "相关" → "双变量"
2. 将math和chinese选入"变量"
3. 勾选"皮尔逊"相关系数
4. 点击"确定"

**解读结果**：
- 相关系数r的范围在-1到1之间
- 绝对值越接近1，相关性越强
- 显著性（Sig.）< 0.05表示相关显著

### 步骤5：线性回归分析

我们尝试用数学成绩预测语文成绩：

1. 点击"分析" → "回归" → "线性"
2. 将chinese选入"因变量"
3. 将math选入"自变量"
4. 方法选择"输入"
5. 点击"确定"

**解读结果**：
- 查看"模型摘要"表格，R²表示模型解释力
- 查看"ANOVA"表格，显著性检验模型是否显著
- 查看"系数"表格，得到回归方程

## 11.10 常见问题

### Q1: 如何保存SPSS数据文件？

A: 点击"文件" → "保存"或"另存为"，选择保存位置，输入文件名。SPSS数据文件的扩展名为.sav。

### Q2: 如何导出分析结果？

A: 在输出窗口中，右键点击需要导出的内容，选择"导出"，选择导出格式（Word、Excel、PDF等）。

### Q3: 中文乱码怎么办？

A: 
1. 确保系统区域设置为中文
2. 在SPSS中点击"编辑" → "选项" → "语言"，设置界面语言和输出语言
3. 导入数据时注意文件编码

### Q4: 如何处理缺失值？

A: 
1. 在变量视图中定义缺失值
2. 使用"转换" → "替换缺失值"进行填充
3. 分析时可以选择排除缺失值

### Q5: SPSS和Excel有什么区别？

A: 
- Excel主要用于数据处理和简单图表
- SPSS专注于专业统计分析
- SPSS提供更丰富的统计方法
- SPSS的结果输出更专业规范
