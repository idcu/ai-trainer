## 环境搭建

### Python安装

1. 下载Python：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 验证安装：`python --version`

### Anaconda安装

1. 下载Anaconda：https://www.anaconda.com/products/distribution
2. 安装时勾选 "Add Anaconda to my PATH environment variable"
3. 验证安装：`conda --version`

### 安装数据科学相关库

#### 使用pip安装（仅安装Python）

```bash
# 安装NumPy
pip install numpy

# 安装Pandas
pip install pandas

# 安装Excel文件处理库
pip install openpyxl  # 处理.xlsx文件
pip install xlrd      # 处理.xls文件

# 安装数据可视化库
pip install matplotlib
```

#### 使用conda安装（已安装Anaconda）

```bash
# 激活环境后
conda install numpy pandas openpyxl xlrd matplotlib
```

**注意**：如果使用Anaconda，通常默认已经包含了NumPy和Pandas，但为了确保版本一致，建议重新安装或更新。

## 常见问题

1. **Python中文乱码**：在文件开头添加 `# -*- coding: utf-8 -*-`
2. **模块导入错误**：使用 `pip install 模块名` 安装缺失的模块
3. **路径问题**：确保文件路径正确，使用绝对路径或相对路径
4. **权限问题**：以管理员身份运行命令提示符
