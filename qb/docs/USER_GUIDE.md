# 使用手册

## 1. 快速开始

### 1.1 访问应用

打开浏览器，访问以下任一地址：
- GitHub Pages: `https://<your-username>.github.io/<repo-name>/`
- Gitee Pages: `https://<your-username>.gitee.io/<repo-name>/`
- 本地: 直接打开 `index.html` 文件

### 1.2 主界面介绍

进入应用后，您将看到主菜单，包含以下功能入口：

- **判断题、单选题、多选题**: 三种题型的快速开始
- **错题本**: 查看和练习错题
- **收藏夹**: 管理收藏的题目
- **历史记录**: 查看答题历史
- **搜索题目**: 搜索和筛选题目
- **混合练习**: 多种题型混合练习
- **统计分析**: 查看学习数据和建议
- **数据管理**: 导入/导出数据

## 2. 基础答题

### 2.1 开始答题

1. 在主菜单选择一种题型（判断题、单选题或多选题）
2. 系统会随机加载题目
3. 开始答题！

### 2.2 答题界面说明

答题界面包含以下元素：

- **顶部信息栏**: 显示当前题号、总题数、当前得分
- **进度条**: 直观显示答题进度
- **题目内容**: 当前题目的文字描述
- **选项区域**: 可点击的选项（单选/多选）
- **操作按钮**: 提交答案、下一题等
- **收藏按钮**: 收藏当前题目
- **计时器** (考试模式): 显示剩余时间

### 2.3 选择答案

- **判断题**: 点击"对"或"错"
- **单选题**: 点击选择一个选项
- **多选题**: 可以点击选择多个选项，再次点击取消选择

### 2.4 提交答案

选择答案后，点击"提交答案"按钮：

- **练习模式**: 立即显示对错、正确答案和解析
- **考试模式**: 不显示答案，直接进入下一题

### 2.5 查看结果

答完所有题目后，会显示结果页面：

- 最终得分
- 正确题数
- 错误题数
- 正确率百分比

## 3. 练习模式与考试模式

### 3.1 练习模式 (默认)

- 答题后立即查看反馈
- 显示正确答案
- 显示题目解析（如果有）
- 可以点击"下一题"继续

### 3.2 考试模式

- 交卷前不显示答案
- 支持时间限制
- 时间到自动交卷
- 交卷后统一查看成绩

### 3.3 切换模式

1. 在主菜单找到"模式选择"
2. 点击"练习模式"或"考试模式"
3. 考试模式下可以设置时间限制（10/20/30/60分钟）

## 4. 错题本功能

### 4.1 自动记录错题

答错的题目会自动添加到错题本中，无需手动操作。

### 4.2 查看错题本

1. 在主菜单点击"错题本"
2. 查看所有错题列表
3. 每个错题显示：
   - 题型标识
   - 错误次数
   - 错误时间
   - 题目内容
   - 正确答案
   - 题目解析

### 4.3 错题重练

1. 在错题本页面点击"开始错题练习"
2. 系统会加载所有错题进行练习
3. 答对的题目不会自动从错题本移除（需要手动移除）

### 4.4 管理错题

- **移除单题**: 点击错题卡片上的"移除错题"按钮
- **清空错题本**: 点击"清空错题本"按钮（此操作不可恢复！）

## 5. 收藏夹功能

### 5.1 收藏题目

在答题时，点击题目旁边的心形按钮：
- 白色心形 (🤍): 未收藏
- 红色心形 (❤️): 已收藏

### 5.2 查看收藏夹

1. 在主菜单点击"收藏夹"
2. 查看所有收藏的题目
3. 每个收藏显示：
   - 题型标识
   - 收藏时间
   - 题目内容
   - 答案和解析

### 5.3 收藏练习

1. 在收藏夹页面点击"开始收藏练习"
2. 系统会加载所有收藏的题目进行练习

### 5.4 管理收藏

- **取消收藏**: 点击收藏卡片上的"取消收藏"按钮，或答题时点击心形按钮
- **清空收藏夹**: 点击"清空收藏夹"按钮（此操作不可恢复！）

## 6. 历史记录

### 6.1 查看历史记录

1. 在主菜单点击"历史记录"
2. 查看所有答题记录
3. 每条记录显示：
   - 练习类型
   - 完成时间
   - 得分
   - 正确数
   - 错误数
   - 正确率

### 6.2 历史统计

在历史记录页面顶部，可以看到：

- 总测试次数
- 平均得分
- 平均正确率
- 最佳成绩

### 6.3 管理历史记录

- **删除单条**: 点击记录上的"删除"按钮
- **清空历史**: 点击"清空历史记录"按钮（此操作不可恢复！）

## 7. 搜索与筛选

### 7.1 搜索题目

1. 在主菜单点击"搜索题目"
2. 在搜索框输入关键词
3. 系统会实时显示匹配的题目
4. 搜索结果中关键词会高亮显示

### 7.2 题型筛选

- 点击"全部题型": 显示所有题目
- 点击"判断题": 只显示判断题
- 点击"单选题": 只显示单选题
- 点击"多选题": 只显示多选题

### 7.3 从搜索结果开始

在搜索结果中：

- 点击"查看答案": 查看该题的答案和解析
- 点击"开始练习": 从该题开始练习该题型

## 8. 混合题型练习

### 8.1 配置混合练习

1. 在主菜单点击"混合练习"
2. 选择要包含的题型（可多选）：
   - 判断题
   - 单选题
   - 多选题
3. 设置题目数量（默认50题）
4. 点击"开始混合练习"

### 8.2 混合答题

- 答题时会显示当前题型标识
- 题目顺序随机
- 包含所有选择的题型

## 9. 答题进度保存

### 9.1 自动保存

答题过程中，系统会自动保存进度：

- 每分钟自动保存一次
- 切换页面时自动保存
- 关闭浏览器前自动保存

### 9.2 恢复进度

如果上次答题未完成，再次打开应用时会提示：

- 显示上次保存的时间
- 点击"恢复进度": 继续上次的答题
- 点击"新练习": 放弃保存的进度，开始新的练习

## 10. 统计与分析

### 10.1 查看统计

1. 在主菜单点击"统计分析"
2. 查看详细的学习数据

### 10.2 统计内容

- **性能等级**: 根据正确率评估的学习水平（入门/初级/中级/高级/专家）
- **学习概览**:
  - 总测试次数
  - 总答题数
  - 答对题数
  - 平均得分
  - 错题数
  - 收藏数
- **题型表现**: 各题型的正确率统计
- **学习建议**: 根据您的表现提供的针对性建议

## 11. 数据管理

### 11.1 导出数据

1. 在主菜单点击"数据管理"
2. 点击"导出数据"
3. 系统会生成一个 JSON 文件并下载
4. 导出的数据包含：
   - 错题本
   - 收藏夹
   - 历史记录
   - 答题进度

### 11.2 导入数据

1. 在主菜单点击"数据管理"
2. 点击"导入数据"
3. 选择或拖拽之前导出的 JSON 文件
4. 确认导入
5. 导入成功后会显示导入摘要

### 11.3 清空数据

1. 在主菜单点击"数据管理"
2. 点击"清空所有数据"
3. 确认清空（此操作不可恢复！）
4. 数据包括：
   - 错题本
   - 收藏夹
   - 历史记录
   - 答题进度
   - 用户设置

## 12. 使用技巧

### 12.1 高效学习建议

1. **先使用练习模式**: 熟悉题目和知识点
2. **善用错题本**: 重点复习错题
3. **收藏重点题**: 标记重要或难题
4. **定期看统计**: 了解自己的薄弱环节
5. **模拟考试**: 偶尔使用考试模式检验学习效果

### 12.2 快捷键

目前版本暂不支持键盘快捷键，后续版本会添加。

### 12.3 数据安全

- 数据存储在浏览器的 LocalStorage 中
- 定期导出数据备份
- 清除浏览器数据会丢失本地数据
- 建议定期导出备份

## 13. 常见问题

### 13.1 题目加载慢？

- 检查网络连接（如果使用在线版本）
- 尝试刷新页面
- 清除浏览器缓存后重试

### 13.2 数据丢失了？

- 检查是否在同一浏览器和设备
- 检查是否清除了浏览器数据
- 如果有导出文件，可以尝试导入恢复

### 13.3 如何更换设备？

1. 在旧设备上导出数据
2. 将导出的文件传输到新设备
3. 在新设备上导入数据

### 13.4 考试模式时间到了会怎样？

- 时间到会自动交卷
- 已答的题目会计分
- 未答的题目不计分

### 13.5 如何反馈问题？

欢迎通过以下方式反馈：
- 提交 Issue
- 发送邮件
- 参与讨论

## 14. 附录

- **判断题**: 2个选项（对/错）
- **单选题**: 4个选项，选择1个正确答案
- **多选题**: 4个选项，选择多个正确答案

| 等级 | 正确率范围 | 说明 |
|------|-----------|------|
| 入门 | 0-59% | 刚开始学习 |
| 初级 | 60-74% | 掌握基础 |
| 中级 | 75-84% | 熟悉内容 |
| 高级 | 85-94% | 熟练掌握 |
| 专家 | 95-100% | 精通 |
